  Formula One Report (1950–2020) — Power BI Dashboard

##  Project Overview
An interactive Power BI dashboard analyzing 70 years of Formula 1 data,
covering drivers, constructors, circuits, and race results from 1950 to 2020.

##  Key Metrics
| Metric | Value |
|--------|-------|
| Total Drivers | 854 |
| Total Constructors | 211 |
| Total GP Events | 1,035 |
| Unique Circuits | 79 |

##  Data Sources
| File | Description |
|------|-------------|
| `results.csv` | Race results with points, positions |
| `races.csv` | Race events by year and round |
| `drivers.txt` | Driver details and nationalities |
| `constructors.csv` | Constructor/team information |
| `circuits.json` | Circuit locations and coordinates |

##  Tools Used
- **Power BI Desktop** — Data modeling & visualization
- **Power Query** — Data transformation & cleaning

##  Key Insights
- 🇬🇧 British constructors lead all-time wins (407)
-  Lewis Hamilton leads total points (4.1K)
-  British GP is the most frequent race (52 times)
- 🇮🇹 Ferrari has the most podiums among constructors

##  How to Open
1. Download `PowerBI/Formula_One_Analysis.pbix`
2. Open with **Power BI Desktop** (free download from Microsoft)
3. Explore the interactive dashboard!
